## ✅ REGLAS DE ORO

1. **Nombres exactos** → Usa los nombres EXACTOS de las guías
2. **Minúsculas** → Todo en minúsculas (sin mayúsculas)
3. **Sin espacios** → Usa guiones: `foto-ejemplo.png`
4. **Extensión .png** → Preferiblemente PNG (o JPG si cambiás el código)
5. **Carpeta correcta** → Cada imagen en su carpeta

## 🆘 ¿PROBLEMAS?

### Imagen no carga:
1. Verificá el nombre (EXACTO, incluyendo minúsculas)
2. Verificá la carpeta correcta
3. Verificá la extensión (.png)
4. Recargá con Ctrl+F5

### Muchas imágenes no cargan:
1. Verificá que creaste `/public/images/`
2. Reiniciá el servidor de desarrollo
