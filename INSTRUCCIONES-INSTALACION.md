## 📂 PASO 1: CREAR ESTRUCTURA DE CARPETAS

Crea la siguiente estructura dentro de la carpeta `/public/` de tu proyecto:

```
/public/
└── images/
    ├── inicio/
    ├── actividades/
    ├── informacion/
    ├── historia/
    ├── nosotros/
    └── contacto/
```

### Cómo crear las carpetas:

**En Windows:**
1. Abre la carpeta del proyecto
2. Navega a `/public/`
3. Click derecho → Nueva Carpeta → `images`
4. Dentro de `images`, crea las 6 subcarpetas

**En Mac/Linux:**
```bash
cd public
mkdir -p images/{inicio,actividades,informacion,historia,nosotros,contacto}
```

## ⚠️ SOLUCIÓN DE PROBLEMAS

### ❌ Problema: "Una imagen no carga"

**Soluciones:**
1. ✅ Verifica que el nombre del archivo sea **EXACTO** (incluyendo mayúsculas/minúsculas)
2. ✅ Verifica que esté en la **carpeta correcta**
3. ✅ Verifica que la **extensión** sea `.png`
4. ✅ Recarga la página con **Ctrl+F5** (Windows) o **Cmd+Shift+R** (Mac)
5. ✅ Verifica que no haya **espacios** en el nombre del archivo

### ❌ Problema: "Varias imágenes no cargan"

**Solución:**
1. Verifica que creaste la carpeta `/public/images/`
2. Verifica que las subcarpetas estén dentro de `/public/images/`
3. Reinicia el servidor de desarrollo

## 📝 NOTAS ADICIONALES

### Formatos de Imagen Soportados
- ✅ PNG (recomendado)
- ✅ JPG/JPEG
- ⚠️ Si usás JPG, cambiá las extensiones en los nombres de archivo de `.png` a `.jpg`

### Tamaños Recomendados
- **Hero images:** 1920x1080px mínimo
- **Galería:** 800x600px mínimo
- **Logos:** 500x500px mínimo
- **Timeline/Cards:** 800x600px

### Optimización
Para mejor rendimiento, optimizá tus imágenes antes de subirlas:
- Usa compresión sin pérdida
- Tamaño máximo: 500KB por imagen
- Herramientas recomendadas: TinyPNG, ImageOptim
